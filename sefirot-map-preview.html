SEFIROT MAP PREVIEW — A vs B

Both variants shown with the pentagram revealed and the LBRP gold text band visible — the busiest state of the map. Each sefira label sits beside its venue's lamp position. Variant A uses 2-character Hebrew abbreviations; Variant B uses full names.
A — 2-CHAR ABBREVIATIONS
2-char Hebrew abbreviations beside each visited venue's lamp position. Small gold marker dot anchors each label. Lower visual mass.
B — FULL HEBREW NAMES
Full Hebrew names. More legible to readers of Hebrew, more visual weight on the map. Slightly more atmospheric.
The dot marker next to each Hebrew label anchors it to the venue's "sefira spot" — distinct from the venue lamps (which are the existing radial glows on the map). On the live map, labels would only render for venues you have actually visited; here all ten are shown to demonstrate the maximum load.
